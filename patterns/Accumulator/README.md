# Accumulator Pattern (Паттерн Накопитель)

## Что такое Accumulator?

Accumulator (накопитель) — один из самых популярных паттернов в алгоритмах.

Его идея проста:

1. Создать переменную для хранения результата.
2. Пройтись по всем данным циклом.
3. На каждой итерации обновлять результат.
4. Вернуть итоговое значение.

---

## Общий шаблон

```js
let result = начальное_значение

for (const item of data) {
  result = обновить(result, item)
}

return result
```

---

## Как распознать этот паттерн?

Если в условии встречаются слова:

* посчитать
* найти количество
* найти сумму
* найти максимум
* найти минимум
* вычислить среднее
* подсчитать символы
* определить самый большой элемент

С высокой вероятностью используется Accumulator.

# Алгоритм решения задач через Accumulator

Шаг 1

Определить, что именно нужно накапливать.

Примеры:

* сумму
* количество
* максимум
* минимум

---

Шаг 2

Создать переменную-накопитель.

```js
let result = ...
```

---

Шаг 3

Пройтись по данным циклом.

```js
for (const item of data) {

}
```

---

Шаг 4

Обновлять накопитель.

```js
result = ...
```

---

Шаг 5

Вернуть результат.

```js
return result
```
