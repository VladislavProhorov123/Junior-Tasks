# Reverse a String

## Суть задачи

Нужно вернуть строку в обратном порядке.

Примеры:

```js
"hello" -> "olleh"
"JavaScript" -> "tpircSavaJ"
```

---

## Что проверяет задача

* понимание строк в JavaScript;
* умение работать с индексами и циклами;
* умение строить новый результат;
* базовую алгоритмическую мышление.

---

## Важный момент

Строки в JavaScript **неизменяемые**. Символ внутри строки нельзя заменить напрямую.

```js
let str = "hello";
str[0] = "H";
console.log(str); // hello
```

Поэтому решение строится через создание новой строки или массива.

---

## Решение через встроенные методы

```js
function reverseString(str) {
  return str.split('').reverse().join('');
}
```

### Как работает

* `split('')` превращает строку в массив символов;
* `reverse()` разворачивает массив;
* `join('')` собирает строку обратно.

Пример:

```js
"hello".split('')
// ['h', 'e', 'l', 'l', 'o']

['h', 'e', 'l', 'l', 'o'].reverse()
// ['o', 'l', 'l', 'e', 'h']

['o', 'l', 'l', 'e', 'h'].join('')
// "olleh"
```

---

## Решение через цикл

```js
function reverseString(str) {
  let result = '';

  for (let i = str.length - 1; i >= 0; i--) {
    result += str[i];
  }

  return result;
}
```

### Логика

Идём по строке с конца к началу и последовательно добавляем символы в новую строку.

---

## Сложность

* Время: **O(n)**
* Память: **O(n)**

Где `n` — длина строки.

---

## Что могут спросить на собеседовании

### Почему нельзя изменить строку напрямую?

Потому что строки в JavaScript неизменяемые.

### Почему используется `split('')`?

Чтобы преобразовать строку в массив символов и удобно с ним работать.

### Какой способ лучше?

* Для короткого кода — встроенные методы.
* Для демонстрации понимания — цикл.

### Что будет с пустой строкой?

Вернётся пустая строка.

### Какая сложность решения?

`O(n)` по времени и `O(n)` по памяти.

---

## Частые ошибки

* попытка изменить строку по индексу;
* отсутствие `return`;
* обход строки с начала, а не с конца.

---

## Что нужно уметь сказать

> Я прохожу строку с конца к началу, собираю новый результат и возвращаю его. Строка в JavaScript неизменяемая, поэтому меняю не исходную строку, а создаю новую. Сложность решения — O(n).

---

## Итог

Для `Reverse a String` нужно помнить три вещи:

* строка неизменяемая;
* реверс можно сделать встроенными методами или через цикл;
* базовая сложность — `O(n)`.
